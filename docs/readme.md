# Drive Integration Demo

A Python package demonstrating basic drive data integration and analysis using NumPy, Pandas, and Matplotlib.

## Features

- Generate synthetic drive data (velocity, acceleration, distance)
- Perform statistical analysis
- Create comprehensive visualizations
- Easy-to-use class-based interface
